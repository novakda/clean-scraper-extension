RESPONSIVE BEHAVIOR:

Width: 800px (fixed)
- List panel: Flexible (400px when detail open, 800px when closed)
- Detail panel: 400px (fixed when open)

Height: 600px (fixed)
- Header: ~60px
- Filters: ~80px
- Content: ~460px (scrollable)


INTERACTIONS:

Click Request Row → Show detail panel on right
Click Detail Close → Hide detail panel
Click Filter Input → Filter updates in real-time
Click Clear Filters → Reset all filters
Click Refresh → Fetch latest requests from background
Click Clear All → Confirm dialog → Clear all requests
Click Auto-Refresh → Toggle 2-second polling
Click Copy Button → Copy data to clipboard
Switch Tabs → Show Request or Response view


KEYBOARD NAVIGATION:

Tab → Focus next element
Shift+Tab → Focus previous element
Enter → Activate button/select item
Escape → Close detail panel (could be added)
Arrow Keys → Navigate list (could be added)
